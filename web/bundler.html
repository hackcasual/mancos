MANCOS BUNDLER

← back to planner

Builds a self-contained bundle from your own copy of Factorio (the parsed prototype database, extracted icons, and locale text) that the planner can load offline — it never reads game files itself. Nothing is uploaded: your data folder is read directly in this tab and the bundle is produced entirely on your machine.

This browser can't grant folder access. The bundler needs the File System Access API (showDirectoryPicker), currently available in Chromium-based browsers (Chrome, Edge). Open this page in Chrome or Edge, or build a bundle with the yafc_bundler command-line tool instead.
Pick your Factorio folders
Data folder (not picked)
Mods folder (vanilla — no mods folder)

Data folder is Factorio's install data/ directory (contains core/ and base/). Mods folder is your Factorio mods/ directory (contains mod-list.json) — leave unpicked to bundle vanilla only. Access is granted for this browser tab only.
